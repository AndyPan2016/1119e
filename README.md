## 项目运行


```
git clone 

cd 1119E2

npm install (安装依赖包)

npm run dev (运行本地开发环境)

npm run build (打包生成代码)

npm run clean (清除所有生成代码)

```

因项目流程全部基于gulp实现，需要在运行打包命令前需要先进行全局gulp安装

```
npm i -g gulp
```
